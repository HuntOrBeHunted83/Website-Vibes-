Sound to Mesh!  

High Level Goals 
- Analyze the wav file and get all of the data from it
- Feed those data values into blender and change the floor values

Specifc Goals
- Goal 1
    - Read the wave file and print the raw metrics
    - Convert those 3 metrics to 3 different kinds of sounds
        - Bass, Treble, Mid
    - Print out those 3 new metrics 
-Goal 2
    - Take the per second avarage of those 3 values 
    - Based on the per second avarage of the values update the image

-Potenial Add Ons
    - Allow a mp3 file to wav file converter
